# Glimpse AI Transformation Widget

This Theme App Extension provides customers with AI-powered product visualizations directly on product pages.

## How It Works

1. **Merchants configure products** in the app admin to enable transformations
2. **Merchants add the widget** to their product pages via the theme editor
3. **Customers upload photos** and see instant AI transformations
4. **Conversion boost** from customers seeing the product effect before buying

## Installation for Merchants

1. Install the Glimpse app
2. Configure products with transformation prompts in the app admin
3. Go to your theme editor (Online Store > Themes > Customize)
4. Navigate to a product page
5. Add the "Glimpse AI Transformation" block to your product page
6. Customize the widget title and description
7. Save and publish

## Widget Features

- **Smart Product Detection**: Automatically works with configured products
- **Beautiful UI**: Modern, mobile-responsive design that fits any theme
- **Before/After Comparison**: Clear visualization of transformation effects
- **Instant Add to Cart**: Seamless integration with existing cart functionality
- **Error Handling**: Graceful fallbacks for network issues
- **Multi-language Support**: Built-in translation support

## Technical Requirements

- Online Store 2.0 compatible theme
- Product pages must support app blocks
- Products must be configured in the app admin first

## Supported File Types

- All image formats (JPEG, PNG, WebP, HEIC, etc.)
- Maximum file size: 5MB

## Browser Support

- Modern browsers with JavaScript enabled
- Mobile and desktop responsive
- Works in theme editor preview mode
