# Glimpse AI Transformation Widget

This Theme App Extension provides customers with AI-powered product visualizations directly on product pages through a beautiful static widget design.

## How It Works

1. **Merchants configure products** in the app admin to enable transformations
2. **Merchants add the widget** to their product pages via the theme editor
3. **Customers interact with the static widget** to upload photos and see instant AI transformations
4. **Conversion boost** from customers seeing the product effect before buying

## Widget Design

The widget features a modern, static rectangular design that integrates seamlessly into product pages:

- **Centered AI Preview Badge**: Distinctive gradient badge with enhanced styling
- **Engaging Header**: "See My Glowup!" title with gradient description text and decorative star icon
- **Interactive Upload Area**: Click or drag-and-drop image upload with placeholder preview
- **Processing State**: Elegant loading animation during transformation
- **Before/After Results**: Side-by-side comparison with gradient "Me" and "Future Me" labels
- **Action Buttons**: Centered Add to Cart button above Retry button with Lucide icons
- **Clean Footer**: Minimalist disclaimer with professional Lucide info icon

## Installation for Merchants

1. Install the Glimpse app
2. Configure products with transformation prompts in the app admin
3. Go to your theme editor (Online Store > Themes > Customize)
4. Navigate to a product page
5. Add the "Glimpse AI Transformation" block to your product page
6. Customize colors, text, and placeholder images in the theme editor
7. Save and publish

## Customization Options

### Content Settings
- **Badge Text**: Customize the AI Preview badge text
- **Widget Title**: Set your own heading (default: "See My Glowup!")
- **Description**: Personalize the widget description (supports gradient styling)
- **Button Text**: Customize the upload button text

### Visual Customization
- **Primary Gradient**: Two-color gradient for badges and labels (Me/Future Me)
- **Button Gradient**: Two-color gradient for all action buttons
- **Caption Gradient**: Two-color gradient for description text
- **Icon Color**: Color for star and other decorative icons
- **Background Color**: Widget background color
- **Text Color**: Main text color for titles
- **Placeholder Image**: Upload custom before/after preview image

## Technical Requirements

- Online Store 2.0 compatible theme
- Product pages must support app blocks
- Products must be configured in the app admin first
- Modern browsers with JavaScript enabled
